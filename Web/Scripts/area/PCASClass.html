介绍

PCASClass,省、市、地区联动选择JS封装类，具有兼容性好、效率高、设置方便等特点，是网站前端程序开发不可多得的省市联动好插件。

示例

  • 省市联动应用实例演示
    • 出生地：
      <script type="text/javascript">new PCAS("P1,请选择省份","C1,请选择城市");</script>
    • 所在地：
      <script type="text/javascript">new PCAS("P2=吉林省,请选择省份","C2,请选择城市");</script>
    • 工作地：
      <script type="text/javascript">new PCAS("P3=江苏省,请选择省份","C3=苏州市,请选择城市");</script>
  • 省市地区联动应用实例演示
    • 出　　 生 　地：

    • 户 口 所 在 地：
      <script type="text/javascript">new PCAS("province1=吉林省,请选择省份","city1=吉林市,请选择城市","area1=龙潭区,请选择地区");</script>

使用建议：

  • 对于非万国码数据，js需正确设置charset属性，请在页面头部用如下方式引用<scrpt src="/path/PCASClass.js" charset="gb2312"></scrpt>
  • 初始数据请在文档加载完成后动态处理，或者将设置的JS脚本放置到对应ID或Name元素的下方，否则引起报错;
  • JS未做容错处理，请核实要设置的ID或NAME页面元素是否存在

配置

PCASClass相关参数如下：

属性 类型 默认 描述
Province ID string/array 必需 省份ID/NAME，通过new PCAS("")第一个参数指定ID
City ID string/array 必需 城市ID/NAME，通过new PCAS("")第二个参数指定ID
Area ID string/array 可选 地区ID/NAME，通过new PCAS("")第三个参数指定ID，若二级联动可省略
SetValue 内置方法 通过如var mypcas=new PCAS()初始化后可动态改变设置数据。如：mypcas.SetValue("吉林省","松原市","宁江区");或者mypcas.SetValue()重新初始化数据;
